## 1. Разложение  $x^7 - 1$  над  $\mathbb{F}_2$
$x^7 - 1 = x^7 + 1$
x = 1 - корень: 1 + 1 = 0

$x^7 + 1 = (x + 1)(x^6 + x^5 + x^4 + x^3 + x^2 + x + 1)$

Пусть  $x^6 + x^5 + x^4 + x^3 + x^2 + x + 1$ = $(x^3 + ax^2 + bx + 1)(x^3 + cx^2 + dx + 1)$  =  
$x^6 + (a+c)x^5 + (ac+b+d)x^4 + (ad+bc+1)x^3 + (bd+a+c)x^2 + (b+d)x + 1$

$$
\begin{cases}
a + c = 1\\
ac + b + d = 1\\
ad + bc + 1 = 1\\
bd + a + c =1\\
b + d = 1
\end{cases}
\Rightarrow
\begin{cases}
c = a + 1\\
a(a+1) + b + d = 1\\
a(a+1) + 1 = 1\\
a^2 + a + 1 =1
\end{cases}
$$

Случай a = 0: c = 1, b = 0, d = 1\
Случай a = 1: c = 0, d = 0, b = 1
$\Rightarrow (x^3 + 1)(x^3 + x^2 + x + 1)$\
Тогда $x^7 + 1 = (x + 1)(x^3 + 1)(x^3 + x^2 + x + 1)$
## 2. Построение 6 порождающих многочленов
Обозначим:
- $m_1(x) = x + 1$
- $m_2(x) = x^3 + x + 1$
- $m_3(x) = x^3 + x^2 + 1$\
Порождающий многочлен $g(x)$ циклического кода длины $n = 7$ над $\mathbb{F}_2$ должен быть делителем $x^7 + 1$. 
Все нетривиальные делители (степень от 1 до 6) получаются как все возможные произведения $m_1^{a_1} m_2^{a_2} m_3^{a_3}$, 
где $a_i \in \{0,1}$, исключая случай $a_1 = a_2 = a_3 = 0$ (многочлен 1, дающий тривиальный код) и $a_1 = a_2 = a_3 = 1$ 
(даёт $x^7 + 1$, что соответствует коду из одного нулевого слова, не используется).
Получаем 6 порождающих многочленов:

| № | Порождающий многочлен $g(x)$ | Степень |
|---|--------------------------------|---------|
| 1 | $m_1 = x + 1$ | 1 |
| 2 | $m_2 = x^3 + x + 1$ | 3 |
| 3 | $m_3 = x^3 + x^2 + 1$ | 3 |
| 4 | $m_1 m_2 = (x + 1)(x^3 + x + 1) = x^4 + x^3 + x^2 + 1$ | 4 |
| 5 | $m_1 m_3 = (x + 1)(x^3 + x^2 + 1) = x^4 + x^3 + x + 1$ | 4 |
| 6 | $m_2 m_3 = (x^3 + x + 1)(x^3 + x^2 + 1) = x^6 + x^5 + x^4 + x^3 + x^2 + x + 1$ | 6 |
## 3. Характеристики полученных кодов
Для каждого порождающего многочлена $g(x)$ определяем параметры кода:
- Длина: $n = 7$
- Размерность: $k = 7 - \deg(g)$
- Минимальное расстояние $d$ определяем с помощью границы БЧХ или известных свойств.
### №1: $g(x) = x + 1$
- Корни: $x = 1$ (в поле $\mathbb{F}_2$)
- Это код с проверкой на чётность.
- Параметры: $[7, 6, 2]$

### №2: $g(x) = x^3 + x + 1$
- Корни в поле $\mathbb{F}_8$: $\alpha, \alpha^2, \alpha^4$, где $\alpha^3 = \alpha + 1$
- Содержит два последовательных корня $\alpha, \alpha^2$ ⇒ $d \ge 3$
- Размерность $k = 4$
- Это код Хэмминга $[7, 4, 3]$

### №3: $g(x) = x^3 + x^2 + 1$
- Корни в поле $\mathbb{F}_8$: $\beta, \beta^2, \beta^4$, где $\beta^3 = \beta^2 + 1$
- Аналогично содержит два последовательных корня ⇒ $d \ge 3$
- Размерность $k = 4$
- Это также код Хэмминга $[7, 4, 3]$, эквивалентный предыдущему

### №4: $g(x) = x^4 + x^3 + x^2 + 1 = m_1 m_2$
- Корни: $1$ (из $m_1$) и $\alpha, \alpha^2, \alpha^4$ (из $m_2$)
- Содержит три последовательных корня: $1, \alpha, \alpha^2$ (в показателях степени: 0, 1, 2) ⇒ $d \ge 4$
- Размерность $k = 3$
- Это симплекс-код $[7, 3, 4]$ (дуальный к коду Хэмминга)

### №5: $g(x) = x^4 + x^3 + x + 1 = m_1 m_3$
- Корни: $1$ и $\beta, \beta^2, \beta^4$
- Аналогично содержит три последовательных корня ⇒ $d \ge 4$
- Размерность $k = 3$
- Это также симплекс-код $[7, 3, 4]$, эквивалентный предыдущему

### №6: $g(x) = x^6 + x^5 + x^4 + x^3 + x^2 + x + 1 = m_2 m_3$
- Корни: все ненулевые элементы поля $\mathbb{F}_8$, кроме 1 (т.е. $\alpha, \alpha^2, \alpha^3, \alpha^4, \alpha^5, \alpha^6$)
- Размерность $k = 1$
- Единственные кодовые слова: все нули и все единицы
- Минимальное расстояние $d = 7$
- Это код с повторением $[7, 1, 7]$
