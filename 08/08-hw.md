# Задача 1
Мультипликативная группа поля $GF(5^2)=GF(25)$ состоит из всех ненулевых элементов, поэтому в ней $25-1=24$ элемента.
Так как $x$ — примитивный элемент, то все ненулевые элементы поля имеют вид

$$
1,\ x,\ x^2,\dots,x^{23},
\qquad x^{24}=1.
$$

Для элемента $x^k$ нужно найти наименьшее $m$, при котором $(x^k)^m=1.$
Это равносильно условию, что число $24$ делит $km$.
Рассмотрим элементы по очереди:
- Для $1=x^0$:
  $1^1=1,$
  поэтому порядок равен $1$.
- Для $x$:
  $x^1\ne 1,\;x^2\ne 1,\;\dots,\;x^{23}\ne 1,\quad x^{24}=1,$
  значит, порядок равен $24$.

- Для $x^2$:
  $(x^2)^{12}=x^{24}=1,$
  и меньшая степень не подходит, значит порядок равен $12$.
- Для $x^3$:
  $(x^3)^8=x^{24}=1,$
  поэтому порядок равен $8$.
- Для $x^4$:
  $(x^4)^6=x^{24}=1,$
  значит, порядок равен $6$.
- Для $x^5$:
  $(x^5)^{24}=x^{120}=(x^{24})^5=1.$
  Меньшая степень не даёт $1$, поэтому порядок равен $24$.
- Для $x^6$:
  $(x^6)^4=x^{24}=1,$
  значит, порядок равен $4$.
- Для $x^7$:
  $(x^7)^{24}=x^{168}=(x^{24})^7=1,$
  поэтому порядок равен $24$.
- Для $x^8$:
  $(x^8)^3=x^{24}=1,$
  значит, порядок равен $3$.
- Для $x^9$:
  $(x^9)^8=x^{72}=(x^{24})^3=1,$
  поэтому порядок равен $8$.
- Для $x^{10}$:
  $(x^{10})^{12}=x^{120}=(x^{24})^5=1,$
  значит, порядок равен $12$.
- Для $x^{11}$:
  $(x^{11})^{24}=x^{264}=(x^{24})^{11}=1,$
  поэтому порядок равен $24$.
- Для $x^{12}$:
  $(x^{12})^2=x^{24}=1,$
  значит, порядок равен $2$.
Далее аналогично получаем:
- $x^{13}$ имеет порядок $24$;
- $x^{14}$ имеет порядок $12$;
- $x^{15}$ имеет порядок $8$;
- $x^{16}$ имеет порядок $3$;
- $x^{17}$ имеет порядок $24$;
- $x^{18}$ имеет порядок $4$;
- $x^{19}$ имеет порядок $24$;
- $x^{20}$ имеет порядок $6$;
- $x^{21}$ имеет порядок $8$;
- $x^{22}$ имеет порядок $12$;
- $x^{23}$ имеет порядок $24$.
## Ответ:
- $1$ имеет порядок $1$;
- $x^{12}$ имеет порядок $2$;
- $x^8,x^{16}$ имеют порядок $3$;
- $x^6,x^{18}$ имеют порядок $4$;
- $x^4,x^{20}$ имеют порядок $6$;
- $x^3,x^9,x^{15},x^{21}$ имеют порядок $8$;
- $x^2,x^{10},x^{14},x^{22}$ имеют порядок $12$;
- $x,x^5,x^7,x^{11},x^{13},x^{17},x^{19},x^{23}$ имеют порядок $24$.
# Задача 2
Рассмотрим 4-разрядный регистр с обратной связью, заданный полиномом
$h(x)=1+x+x^4.$
Будем считать, что состояние регистра в момент $t$ имеет вид
$(a_t,a_{t+1},a_{t+2},a_{t+3}).$
Тогда из полинома $h(x)=1+x+x^4$ получаем рекуррентное соотношение
$a_{t+4}=a_{t+1}\oplus a_t,$
где $\oplus$ — сложение по модулю 2.

---
## 1. Смена состояний регистра при начальном заполнении $0,0,1,1$
Начальное состояние:
$(0,0,1,1).$
Дальше каждое следующее состояние получаем сдвигом влево и добавлением нового бита
$a_{t+4}=a_{t+1}\oplus a_t.$
Получаем цикл:
$$(0,0,1,1)\to(0,1,1,0)\to(1,1,0,1)\to(1,0,1,0)\to(0,1,0,1)\to(1,0,1,1)\to(0,1,1,1)\to(1,1,1,1)\to(1,1,1,0)\to(1,1,0,0)\to(1,0,0,0)\to(0,0,0,1)\to(0,0,1,0)\to(0,1,0,0)\to(1,0,0,1)\to(0,0,1,1).$$

## 2. Почему это регистр максимальной длины

Здесь регистр имеет $4$ разряда, поэтому максимальный возможный период ненулевой последовательности равен
$2^4-1=15.$
Из полученной цепочки видно, что до возврата в начальное состояние проходит ровно $15$ различных ненулевых состояний.
Следовательно, период равен $15$, то есть он максимален.
Значит, полином $h(x)=1+x+x^4$ является примитивным, а данный регистр — **регистр максимальной длины**.
## 3. Минимальное расстояние кода
Так как задан $(15,4)$-код с проверочным полиномом $h(x)$, это циклический код длины $15$ и размерности $4$.
Его порождающий полином равен
$g(x)=\frac{x^{15}+1}{h(x)}=1+x+x^2+x^3+x^5+x^7+x^8+x^{11}.$
Это код симплекса, двойственный к коду Хэмминга $(15,11,3)$.  
Для него все ненулевые кодовые слова имеют вес $8$.
Следовательно, минимальное расстояние равно
$d_{\min}=8.$
## Ответ
- смена состояний регистра при начальном заполнении $0,0,1,1$:
$(0,0,1,1)\to(0,1,1,0)\to(1,1,0,1)\to(1,0,1,0)\to(0,1,0,1)\to(1,0,1,1)\to(0,1,1,1)\to(1,1,1,1)\to(1,1,1,0)\to(1,1,0,0)\to(1,0,0,0)\to(0,0,0,1)\to(0,0,1,0)\to(0,1,0,0)\to(1,0,0,1)\to(0,0,1,1);$
- период равен $15=2^4-1$, значит это регистр максимальной длины;
- минимальное расстояние кода:
$d_{\min}=8.$
