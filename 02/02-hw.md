# ДЗ 2
Требуется показать, что дуальный код кода Хэмминга является симплекс-кодом, то есть расстояние между любыми двумя различными кодовыми словами равно $2^{r-1}$.

Рассмотрим код Хэмминга длины $n = 2^r - 1$ и размерности $k = 2^r - 1 - r$. Его дуальный код имеет параметры $(2^r - 1, r)$.

Поскольку код линейный, достаточно доказать, что вес любого ненулевого кодового слова дуального кода равен $2^{r-1}$. Тогда минимальное расстояние $d = 2^{r-1}$, и все расстояния между различными словами будут одинаковыми.

Кодовые слова дуального кода — это линейные комбинации строк проверочной матрицы Хэмминга. Столбцы этой матрицы — все ненулевые векторы пространства $\mathbb{F}_2^r$.

Для любого ненулевого вектора $m \in \mathbb{F}_2^r$ координаты кодового слова $c$ вычисляются как $c_j = \langle m, h_j \rangle$, где $h_j\$ — $j$-й столбец.

Для фиксированного ненулевого $m$ скалярное произведение $\langle m, v \rangle = 1$ выполняется ровно для половины векторов $v$ из $\mathbb{F}_2^r$ (т.е. для $2^{r-1}$ векторов). Следовательно, в каждом ненулевом кодовом слове ровно $2^{r-1}$ единиц.

Таким образом, все ненулевые кодовые слова имеют вес $2^{r-1}$, что означает: дуальный код Хэмминга — симплекс-код с постоянным весом и постоянным расстоянием $2^{r-1}$.
# ДЗ 3

Рассмотрим **(n, n-1)**-код с проверкой на четность. Тогда **дуальный ему код** (dual code) будет иметь параметры **(n, 1)** — это код с повторением.

## Матричное представление

**Порождающая матрица** G дуального кода имеет вид строки из n единиц:

$$G = \begin{bmatrix}1 & 1 & \dots & 1\end{bmatrix}$$

**Проверочная матрица** H этого кода:

$$H = \begin{bmatrix}I_{n-1} & | & \mathbf{1}\end{bmatrix}$$

где $I_{n-1}$ — единичная матрица размера $(n-1) \times (n-1)$, а $\mathbf{1}$ — вектор-столбец из единиц.

## Кодовые слова

Так как размерность дуального кода $k = 1$, количество информационных слов составляет $2^1 = 2$ (биты 0 и 1). Следовательно, кодовых слов тоже два:
- Для информационного бита 0: $(0, 0, \dots, 0)$
- Для информационного бита 1: $(1, 1, \dots, 1)$

## Расстояние между кодовыми словами

Расстояние Хэмминга между этими двумя кодовыми словами равно $n$, так как они различаются во всех позициях. Это расстояние является **минимальным расстоянием кода**:
$$d_{\min} = n$$

Поскольку в дуальном коде всего два кодовых слова, это расстояние является не только минимальным, но и единственным расстоянием между любыми различными кодовыми словами.

## Корректирующая способность

Для кода с минимальным расстоянием $d_{\min}$ количество исправляемых ошибок определяется формулой:
$$t = \left\lfloor \frac{d_{\min} - 1}{2} \right\rfloor$$
